# @amlplugins/twilio-verify

Native `twilio` SDK re-exported under the `@amlplugins` namespace.

> Twilio Verify — 2FA via SMS, voice, email, TOTP, Push, Silent Network Auth.

## Install

```bash
npm install @amlplugins/twilio-verify
```

## Use

```ts
import { /* named exports from twilio */ } from "@amlplugins/twilio-verify";
// or
import sdk from "@amlplugins/twilio-verify";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
